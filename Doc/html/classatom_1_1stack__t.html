Detailed Description

template<typename Tp, typename Container = atom::vector_t<Tp>>
class atom::stack_t< Tp, Container >

Template Parameters
TpThe type of the value in the stack
Containerthe type of container which houses the stack (default is atom::vector_t)

Container must have function: push_back(), erase(), back(), size(), swap() and capacity()
